What is «IsRespo»?

Ce este «IsRespo»?

IsRespo is a simple tool that allows us to test a webpage resposivity on different devices: desktops, tablets, smartphones and so on.

The application is open source published on GitHub.

IsRespo este o unealtă simplă și rapidă care ne ajută ca să testăm responsivitatea unei pagini sau aplicații web pe diferite dispozitive: desktop-uri, tablete, telefoane mobile etc.

Aplicația este cu sursă deschisă și publicată pe GitHub.

Simple and Beautiful

Simplu și frumos

Combinând designul flat cu puterea limbajelor de prezentare HTML5 și CSS3, IsRepo este o unealtă perfectă pentru a simula compatibiltatea paginilor web cu dispozitivele mobile.

Combining the flat design with the power of HTML5 and CSS3, IsRepo is perfect for simulating quickly the resposivity of your web page or web application.
